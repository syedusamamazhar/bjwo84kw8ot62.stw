About EAS



Education Administrative System (EAS) is an online management system for those schools where administrative department lack in managing students and teachers record. It comprises of different types of modules which could be purchased separately as per school requirement. For example: if a school only wants a finance module, so there is no need to buy the whole software. They can pay for that module and have access to our finance software. There are 3 modules in EAS:
1. Admin Module
2. Academic Module
3. Finance Module
